# OBJ-Files

Dieses Repository enthält OBJ-Dateien und ein sofort testbares Forge-Beispielprojekt.

## Enthalten
- `Destillery/` – Original-Export (`.obj` + `.mtl`)
- `forge-test-1.20.1/` – lauffähige Forge-Testmod für **Minecraft 1.20.1 / Forge 47.0.4**

Wenn du das Modell direkt im Spiel testen willst, nutze:
`forge-test-1.20.1/README.md`

Direkt aus dem Repo-Root starten:
`./gradlew runClient`
